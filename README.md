# 基于 Scratch3 + Python 的RPA工具

1. 快速开发自定义组件，有助于同步学习scratch3和python

2. 操作简便，方便上手，真正无代码

3. 能支持日常工作，如网页抓取、文件读写、微信消息读写、Excel操作（待）等，实用性强，寓实践性于教学之中

4. 支持多窗口、多线程处理

5. 此工具可完全打包成可执行exe文件，方便运行
